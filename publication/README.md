## Key Findings

- **87.8%** of 37,886 tokens decoded (2.1% false positive rate)
- **18** correctly conjugated forms of NOCERE
- **9/86** CI core terms present — oil-only pharmacy, not a general pharmacopoeia
- **95.9%** of CI terms encoded through the cipher appear in the Voynich
- Reverse encoding matches word length (5.1 chars) and token frequencies
- Closest comparandum: Dioscorides *De Materia Medica* Book I (oils, ointments, aromatics)
- Closest contemporary: Pseudo-Mesue *De Oleis* (c. 1270–1300, Avignon/Lyon)
